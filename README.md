# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite. The react-router-dom package is also included.

### Requirements:


- [Node.js](https://nodejs.org/) (>= 20.19.0)


### Running application:
**Install dependencies**
   ```bash
   npm install
   ```
**Start Development Server**
   ```bash
   npm run dev
   ```
